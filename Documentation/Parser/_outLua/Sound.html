Generic Class:

Sound

One or more sound samples that all represent a specific sound effect.

Has no parent class.


Function:

IsPlaying

Indicates whether this Sound is currently being played.

Arguments:
None.

Return value:
Whether this Sound is being played on a channel.


Function:

Stop

Stops playback of this Sound.

Arguments:
None.

Return value:
Whether this Sound successfully stopped playing.


Function:

UpdateDistance

Updates the distance attenuation of the sound while it's playing.

Arguments:
How much distance attenuation to apply to the sound. 0 = full volume
1.0 = max distant, but still won't be completely silent.

Return value:
Whether this Sound's attenuation setting was successful.
